# Dynamic Event Calendar Application

This is a dynamic calendar application built with **React.js** using **Vite**. The app allows users to manage events with features like adding, editing, and viewing events.

## Features
- Display a calendar with proper weekday/weekend alignment.
- Highlight the current day in green.
- Display events for the selected day.
- Add new events with details like:
  - Event name
  - Start and end time
  - Description (optional)
- Persist data using localStorage.

### Bonus Features
- Form with a fade-in effect.
- Responsive layout.
